Commands

Commands For Crow discord Bot

User

/user_balance: Shows the bot balance of the user.

/user_deposit: Prompts the user with account number and memo to deposit TNBC.

/user_withdraw amount: AMOUNT: Lets the user withdraw TNBC to their withdrawal address.

/user_setwithdrawaladdress account_number: ACCOUNT_NUMBER Lets the user set/ update the withdrawal address.

Escrow

/escrow_add amount: AMOUNT user: DISCORD_USER: Lets user create escrow ticket.

/escrow_status escrow_id: ESCROW_ID: View the status of the escrow with the escrow id.

/escrow_all: Lists all the active escrows of the particular user.

/escrow_release escrow_id: ESCROW_ID: Lets the initiator release the escrow fund.

/escrow_cancel escrow_id: ESCROW_ID: Lets the initiator or successor to cancel the escrow. (Both users will need to use the command to cancel the escrow)

/escrow_dispute escrow_id: ESCROW_ID: Creates a dispute with the escrow id.

Agent

/agent_release escrow_id: ESCROW_ID user: USER: Lets the agent release the fund to particular user.

/agent_cancel escrow_id: ESCROW_ID: Lets the agent cancel the escrow.
