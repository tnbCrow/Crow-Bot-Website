Getting Started

Trade TNBC P2P on tnbCrow discord server!

Your role?

  • I want to Buy TNBC
  • I want to Sell TNBC

  1. Join tnbCrow discord server. Invite Link

  2. Find your trade partner from the #trade channel and close the trade deal.

  3. Ask the seller to create an escrow.

  4. Use the command /escrow all to check the escrow.

  5. Once verified, pay the seller using the payment method you have agreed on.

    6. Warning: Never send any payment before verifying that the escrow exists.

  6. Once paid, ask the seller to release their funds into your account.

    7. Ran into disputes? Use the /escrow dispute escrow_id: ESCROW_ID command and the tnbCrow staff will create a private channel within the tnbcrow server to resolve the issue.

  7. Use /balance to check your new balance.

  1. Join tnbCrow discord server. Invite Link

  2. Find your trade partner from the #trade channel and close the trade deal.

  3. Deposit TNBC into your account by using the /deposit tnbc command.

  4. Create an escrow by using the command /escrow tnbc amount: AMOUNT user: DISCORD_USER.

  5. By using the command /escrow all, ask the buyer to verify the escrow

  6. Once verified, ask the buyer to send the payment.

  7. Once the payment is received, release funds using the command /escrow release escrow_id: ESCROW_ID.

    8. Warning: Never release escrow before verifying that you have received payment.
    Ran into disputes? Use /escrow dispute escrow_id: ESCROW_ID command and the tnbCrow staff will create a private channel in the tnbcrow server to resolve the issue.

  8. Use /balance to check your new balance.

How does escrow work on the crow bot?

  • Buyer and seller must agree to terms and the payment method.

  • The seller creates an escrow ticket. The crow bot locks TNBC in the seller's crow bot account.

  • Buyer verifies the ticket creation by using the command /escrow all to list all active escrows.

  • The buyer sends the payment to the seller.

  • The seller receives the payment and releases the funds to the buyer. If not released, the buyer raises a dispute using the /escrow dispute escrow_id: ESCROW_ID command.
